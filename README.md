\# TD Python Module 4



\## Automatisation métier avec Python



Ce TD présente plusieurs exercices pratiques autour de :



\- l’automatisation de tâches

\- le monitoring de données

\- la gestion des alertes

\- le reporting automatique

\- l’utilisation de SQLite

\- l’analyse de portefeuille avec pandas



\---



\## Exercices



\### Exercice 1

Automatisation de tâches avec schedule.



\### Exercice 2

\### Exercice 3

Détection des positions risquées.



\### Exercice 4

Stockage des données dans SQLite.



\### Exercice 5

Génération automatique de rapports.
